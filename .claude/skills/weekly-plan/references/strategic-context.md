## How to Search

Use `mcp__basic-memory__search_notes` with the document title or key phrases. If a doc isn't found by title, try:
1. Search by folder path
2. Search by key phrases from the document
3. Use `mcp__basic-memory__read_note` with the exact permalink if known

## Scorecard Calculations

- **Completion rate**: completed tasks / total tasks (exclude cancelled)
- **Time invested**: sum of `plannedTime` on completed tasks (in minutes, convert to hours)
- **Backlog age**: calculate days since each backlog item was created, flag >30 days
- **Strategy alignment**: map completed tasks to strategic priorities, identify gaps
